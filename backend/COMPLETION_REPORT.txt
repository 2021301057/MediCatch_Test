================================================================================
                          QUICK START
================================================================================

1. Initialize Database:
   mysql -u root -p < backend/init.sql

2. Start Services (7 terminals):
   Terminal 1: cd eureka-server && gradle bootRun
   Terminal 2: cd gateway-service && gradle bootRun
   Terminal 3: cd user-service && gradle bootRun
   Terminal 4: cd health-service && gradle bootRun
   Terminal 5: cd insurance-service && gradle bootRun
   Terminal 6: export OPENAI_API_KEY=sk-... && cd chat-service && gradle bootRun
   Terminal 7: cd analysis-service && gradle bootRun

3. Verify Services:
   - Eureka: http://localhost:8761
   - Gateway: http://localhost:8000

4. Test Chat API:
   curl -X POST http://localhost:8000/api/chat/message \
     -H "Content-Type: application/json" \
     -H "Authorization: Bearer JWT_TOKEN" \
     -d '{"userId":1,"message":"당뇨병 검사 보험은?"}'

See QUICK_START.md for detailed instructions.
